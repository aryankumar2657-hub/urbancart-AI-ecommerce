# UrbanCart Frontend

React frontend for UrbanCart e-commerce. Includes modern UI, cart, checkout, bill generation and transaction history.

## Commands

```bash
npm install
npm start
```

## Required `.env`

```env
REACT_APP_API_URL=http://localhost:1337/api
REACT_APP_UPLOAD_URL=http://localhost:1337
REACT_APP_API_TOKEN=
```
